sin θ — Sine

sinθ = Opposite / Hypotenuse.
Example: sin(30°)=0.5

cos θ — Cosine

cosθ = Adjacent / Hypotenuse.
Example: cos(60°)=0.5

tan θ — Tangent

tanθ = Opposite / Adjacent.
Example: tan(45°)=1

sin⁻¹ x — Inverse Sine

Gives angle whose sine value is x.
Example: sin⁻¹(0.5)=30°

cos⁻¹ x — Inverse Cosine

Angle whose cosine = x.

tan⁻¹ x — Inverse Tangent

Angle whose tangent = x.

sec θ — Secant

secθ = 1 / cosθ

csc θ — Cosecant

cscθ = 1 / sinθ

cot θ — Cotangent

cotθ = 1 / tanθ

sinh x — Hyperbolic Sine

sinh x = (eˣ − e⁻ˣ) / 2

cosh x — Hyperbolic Cosine

cosh x = (eˣ + e⁻ˣ) / 2

tanh x — Hyperbolic Tangent

tanh x = sinh x / cosh x

sinh⁻¹ x — Inverse Hyperbolic Sine

asinh(x) = ln(x + √(x²+1))

cosh⁻¹ x — Inverse Hyperbolic Cosine

acosh(x) = ln(x + √(x−1)(x+1))

tanh⁻¹ x — Inverse Hyperbolic Tangent

atanh(x)=½ ln((1+x)/(1−x))

log₁₀ x — Common Logarithm

log base 10.
Example: log(100)=2

ln x — Natural Logarithm

Log base e.
ln(e)=1

logₓ y — Log Base x

logₓ(y)= ln(y) / ln(x)

√x — Square Root

Value which squared gives x.

x² — Square

x × x

x³ — Cube

x × x × x

xʸ — Power

x raised to exponent y.

Mod — Remainder

Gives remainder of division.

e — Euler's Number

e = 2.718
Used in exponential growth, logs, calculus.

n! — Factorial

n! = n × (n−1) × ... × 1

d/dx — Differentiation

Derivative gives rate of change.

Examples:
d/dx (x²) = 2x
d/dx (sin x) = cos x

∫ — Integration

Integral gives area under curve.

Examples:
∫ x dx = x²/2
∫ cos x dx = sin x
